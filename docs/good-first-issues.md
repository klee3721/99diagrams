# Good first issues

These issues are intentionally small and independent. They are good candidates
for new contributors after reading `CONTRIBUTING.md`.

1. Add one more starter template using only existing node kinds.
2. Add an e2e smoke path for opening the shortcut help dialog by keyboard.
3. Add malformed `.99draw.json` fixtures for missing page/layer metadata.
4. Improve empty-state copy for first-time users in English and Vietnamese.
5. Add tooltip text for disabled arrange actions.
6. Add a README screenshot generated from an original 99draw demo diagram.
7. Add a small fixture covering dashed bidirectional edges.
8. Add docs for keyboard-only diagram creation.
9. Add a benchmark case for export SVG time on a 1,000-node diagram.
10. Add an issue template for accessibility bugs.
11. Add a minimal architecture ADR for choosing React Flow.
12. Add a self-host guide section for Caddy.
13. Add a test that confirms unsupported image URL imports are rejected.
14. Add a UI status message after successful copy-image.
15. Add a demo `.99draw.json` file for an incident response flow.
16. Add release-note examples for known limitations.
17. Add a contrast regression checklist for new themes.
18. Add a browser benchmark note explaining local-machine variability.
19. Add docs for how to sanitize a sample file before attaching it to an issue.
20. Add a small visual fixture for horizontal swimlane edge labels.
